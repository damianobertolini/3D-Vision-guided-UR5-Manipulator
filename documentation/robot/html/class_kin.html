int  steps = 3000,
double  minT = 0 
)

creates a joint path between 2 points in space

uses path and trajectory cubical interpolation in the joint space
extract the joint position with compute_ik(Eigen::Vector < double, 6 > )
and interpolates with a cubical, and calculates the joint position of each step
coses the shortest path on rotations and constrains the final angle with
constrainAngle180(Eigen::Vector<double, 6>).
returns a standard vector of joint positions

See also
Helper::constrainAngle180(Eigen::Vector<double, 6>)
